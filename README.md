This repository describes the GNU Octave/MATLAB code for Krönig-Kramers analysis of Reflectance spectra of four gemstones: Cubic Zirconia, Moissanite, Quartz and Diamond. 
The code transforms the reflectance to phase and converts it into refractive index and extinction coefficient.
The derived dilectric function is then corrected with doubly subtractive Krönig-Kramers analysis and the output is a plot of dielectric function of these three gemstones
